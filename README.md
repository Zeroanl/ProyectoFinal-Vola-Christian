## Archivos necesarios

### PYTHON
Para comenzar primero tienen que asegurarse que tienen instalado, python.

En windows tiene que abrir una terminal cmd o powershell.
```
PS C:\> python --version
Python 3.X.X 
```
En Linux/Mac tiene que abrir una terminal bash
```
$ python --version
Python 3.X.X 
```
Si les aparece la versiÃ³n todo OK pueden seguir. Caso contrario descarguen python desde este [Link](https://www.python.org/downloads/)

### DJANGO
En una terminal cmd o powershell desde windows:
```
C:\> pip install django
```
Linux/Mac:
```
$ pip install django
```
Si no arrojo errores esto es suficiente para poder correr el projecto.

### DJANGO BOOTSTRAP V5
En una terminal cmd o powershell desde windows:
```
C:\> pip install django-bootstrap-v5
```
Linux/Mac:
```
$ pip install django-bootstrap-v5
```

### DJANGO CRISPY FORMS
En una terminal cmd o powershell desde windows:
```
C:\> pip install django-crispy-forms
```
Linux/Mac:
```
$ pip install django-crispy-forms
```

## Correr el Servidor
Los siguientes comandos son analogos en Mac/Linux/Windows:
```
cd 'direccion carpeta contenedora'
```
Luego arrancamos el servidor web
```
python manage.py runserver
```
Ahora Hace click en el siguiente link para ver el ejemplo corriendo: 
[http://localhost:8000/](http://localhost:8000/)
